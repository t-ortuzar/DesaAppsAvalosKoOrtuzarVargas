📡 Manejo de Estado y Flujos

StateFlow / MutableStateFlow · Manejo reactivo de estados entre capas. Coroutines para operaciones asíncronas (viewModelScope).
ConnectivityObserver monitorea la red en tiempo real. PriceRefreshManager emite actualizaciones progresivas por lotes.

🔧 Inyección de Dependencias

ServiceLocator (Manual DI) · Inicializa y provee todas las dependencias: Database, DAOs, DataStore, Repositories, UseCases, API Services, ConnectivityObserver.
