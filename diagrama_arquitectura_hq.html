📡 Manejo de Estado y Flujos

StateFlow / MutableStateFlow — Manejo reactivo de estados UI.
Coroutines con viewModelScope para operaciones asíncronas.
ConnectivityObserver monitorea la red. PriceRefreshManager emite actualizaciones progresivas por lotes de 5 juegos.

🔧 Inyección de Dependencias

ServiceLocator (Manual DI)
Inicializa y provee: Database, DAOs, DataStore, Repositories, UseCases, API Services y ConnectivityObserver.
Inicializado en GameTrackerApp (Application).

🧪 Testing

~200 tests unitarios con JUnit + Mockito-Kotlin.
Turbine para testing de Flows. JaCoCo: 94% cobertura de instrucciones.
Cubre modelos, use cases, ViewModels, repos y servicios.
